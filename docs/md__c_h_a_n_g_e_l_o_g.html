Changelog

1.2.0

New features

  • PCF85063A / PCF85063TP: timer() now accepts an optional pulse parameter to select the interrupt output mode
// flag mode (default — same behavior as before)
rtc.timer(1.0);
rtc.timer(1.0, false);
// pulse mode
rtc.timer(1.0, true);

API changes (breaking)

The following identifiers contained typos and have been corrected. Replace old names with new names in your sketch.

PCF85263A — enum values

Before (typo) After (corrected)
PCF85263A::INTA_INTTERRUPT PCF85263A::INTA_INTERRUPT
PCF85263A::INTB_INTTERRUPT PCF85263A::INTB_INTERRUPT

PCF85263A — method names

Before (typo) After (corrected)
rtc.pin_congfig(cfg_a, cfg_b) rtc.pin_config(cfg_a, cfg_b)
rtc.ts_congfig(setting) rtc.ts_config(setting)
